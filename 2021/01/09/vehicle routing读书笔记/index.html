Chapter 2: Classical Exact Algorithm for the Capacitated Vehicle Routing Problem

2.1 Introduction

这一章主要介绍Capacitated Vehicle Routing Problem (CVRP)问题的精确算法，主要包括Branch and Bound、column generation、branch and cut。

2.2 Branch and Bound Algorithms

如果从任何点$i$到任何其他点$j$的距离或成本与从$j$到$i$的距离或成本相同，则称旅行商问题是对称(symmetric)的，否则，问题是不对称(asymmetric)的。

ACVRP(asymmetric CVRP) 问题定义为：一个完全有向图 $G=(V,A)$, 顶点表示为 $V=\{0\}\cup N=\{0,1,…,n\}$，其中vertex 0表示depot，其余表示顾客。集合$A$表示有向弧的集合（若是symmetric CVRP问题，则为无向图$G=(V,E)，E$为无向边）。弧的成本矩阵表示为$c$，为非对称矩阵。对于每个顾客$i \in N$都有对应的需求$q_i\geq0$，在depot有包含$K$辆车的车队，车辆容量为$Q$。VRP1和VRP2分别为ACVRP和SCVRP的模型，具体为：
